# SBG-EZLobbyLink
#### _A BepInEx5 mod to easily create a steam lobby code to paste to your about to be former friends_

## Installation

- [Install BepInEx 5](https://github.com/BepInEx/BepInEx/releases) 
- Download and drop the SBG-EZLobbyLink.dll file in the \BepInEx\plugins folder of your Super Battle Golf Installation


## Building for source
*set the env variable ``SUPER_BATTLE_GOLF_PATH`` to your Super Battle Golf installation directory.*
```sh
dotnet build
```
